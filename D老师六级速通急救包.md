## ✍️ 第二部分：反模板写作框架（改卷老师看不出来是模板）

### 核心原则
**不要背整篇范文！** 背**逻辑骨架 + 高级替换词**，考场现场组装。

### 骨架一：议论文（最常见，占50%+）

**第一段：引出话题（2-3句）**
```
第一句（现象/背景）：The past decade has witnessed a growing tendency that ______.
    或：In an era of rapid digital advancement, the issue of ______ has sparked heated debate.

第二句（引出争议）：While some contend that ______, others argue that ______.
    (争议性话题必用这句，不用写观点，只写两方存在)

第三句（亮出立场）：From my perspective, the latter/former stance carries more weight.
```
✅ 反模板秘诀：第一句不用模板套话"In contemporary society"，用 **"The past decade has witnessed..."** 自然引入

**第二段：论证展开（4-5句）**
```
第一句（主题句）：The primary reason why I hold this view lies in ______.

第二句（展开）：To be specific, ______.

第三句（举例/数据）：A case in point is that ______.
    或：According to a recent survey conducted by ______, ______.

第四句（反向让步）：Admittedly, ______ indeed plays a role in certain contexts.
    (承认反方有道理，显你有辩证思维)

第五句（转折拉回）：However, this does not negate the fact that ______.
```
✅ 反模板秘诀：**让步-转折结构**（第四、五句）让文章自然不僵硬

**第三段：结论升华（2句）**
```
第一句（总结）：In conclusion, weighing both sides, I am convinced that ______.

第二句（升华/建议）：Only by striking a balance between ______ and ______ can we ______.
```
✅ 反模板秘诀：用 **"Only by...can we..."** 倒装结尾，语法高级且自然

### 骨架二：图表作文（看图说话）

**第一段：描述图表（2句）**
```
As is vividly illustrated in the column/line/pie chart, ______ witnessed a notable surge/decline between ______ and ______, with ______ reaching a peak/trough at ______ in ______.
```

**第二段：分析原因（4-5句）**
```
Several factors account for this trend. To begin with, ______.
Additionally, ______. Last but not least, ______.
```

**第三段：预测/建议（2句）**
```
Should this trend continue, it is reasonable to anticipate that ______.
Thus, concerted efforts from both individuals and authorities are needed to ______.
```

## 🔄 第三部分：翻译高频句型（背了就能用）

### 句型1：被字句 → 主动变被动
> **中文：** 每年有数百万人参观故宫。
> **六级翻：** The Forbidden City **is visited by** millions of people annually.

> **中文：** 这一政策被广泛认为是成功的。
> **六级翻：** This policy **is widely regarded as** a success.

### 句型2："随着……" → As / With
> **As** the economy develops rapidly, people's living standards have improved dramatically.
> **With** the rapid development of the economy, people's living standards have improved dramatically.

### 句型3："越来越……" → increasingly / ever-growing
> ~~more and more people~~ ❌
> An **increasing** number of people
> **increasingly** important = 越来越重要
> the **ever-growing** popularity of = ……越来越受欢迎

### 句型4："是……的" → 定语从句/分词
> **中文：** 这是一个有着悠久历史的城市。
> **六级翻：** This is a city **that boasts** a long history. (不用which has)
> **中文：** 位于北京中心的故宫是……
> **六级翻：** **Located in** the heart of Beijing, the Forbidden City...

### 句型5："不仅……而且……" → not only倒装
> **Not only** does it boost economic growth, **but it also** enhances cultural exchange.
> ✅ 倒装让阅卷老师眼前一亮

### 句型6："把A看作B" → regard A as B / perceive A as B
> Traditional culture **is perceived as** a vital source of national confidence.

### 句型7："越来越受重视" → gain increasing attention
> Environmental protection **has gained increasing attention** in recent years.

## 🎯 第四部分：蒙题技巧（不会做也能拿分！）

### 📻 听力

**Section A 长对话（8题）：**
- 👉 开头3句必出第一题答案！一定要听到
- 👉 **but / however / actually** 后必出答案
- 👉 **语气突变**处（惊讶/遗憾）必出题
- 👉 听到什么选什么（同义替换多的那个选项）

**Section B 短文（7题）：**
- 👉 **段首段尾**必出题
- 👉 强调语气的词后：**especially / notably / particularly / indeed**
- 👉 **举例处**：for example / such as / like 容易出细节题
- 👉 时间/数字/年代出现时，必考其中一个

**Section C 讲座/讲话（10题）：**
- 👉 注意**逻辑连接词**后：firstly / finally / in addition / moreover
- 👉 **问题-回答结构**：self-asked question = 出题点
- 👉 演讲者**重复强调**的词就是关键词
- 👉 结尾总结句常出最后2题

**🎧 听力终极蒙题法则：**
- 四个选项中，**最长的那个**正确率最高（A/B/C/D里长的优先选）
- 有**绝对词**（only / never / all / must）的选项90%是错的
- 两个意思**相反**的选项，必有一个是对的
- **同义替换**最多的那个选项 = 答案

### 📖 仔细阅读（2篇，10题）

**做题顺序：** 先看题干（不看选项）→ 回原文定位 → 找同义替换

**正确选项特征：**
1. 有**同义替换**（原文词换了说法）
2. **温和委婉**（may / might / could / tend to / likely）
3. **概括性**强（不具体到数字/名字）
4. 和**段落主旨**一致

**错误选项特征：**
1. **原词重现**（原文一模一样的词，往往是陷阱）
2. **绝对化**（only / all / never / must）
3. **张冠李戴**（原文A的特征放在B身上）
4. **无中生有**（完全没提到的信息）
5. **答非所问**（表述正确但不是问的）

**蒙题口诀：**
> 同义替换是答案，原词复现是陷阱
> 委婉选项大概率，绝对选项赶紧扔

### 🔀 段落匹配（15选10，15段）

**核心：不需要完全读懂！**

1. **先看10个题干**，画出关键词（大写/数字/名词短语/生僻词）
2. **回文章快速扫读**，找关键词（同义替换！）
3. 每段**首尾句**重点看（出题点最多）
4. 匹配了就**划掉**，不纠结

**📌 必杀技：**
- 如果一段里有**2个以上题干中的关键词**——大概率是答案
- 段落长 > 匹配可能性大
- 实在找不到的题，先跳过，做完剩下的再回头看
- 剩最后2题时，从没选过的段落里找

### 📝 选词填空（15选10）

**性价比最低！** 最后做！剩多少时间就做多少。

**策略：**
1. 先给15个词**标词性**（n./v./adj./adv.）
2. 看空格前后来判断需要什么词性
3. 认识的词优先选，不认识的看词性蒙
